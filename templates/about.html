About DepositIQ

A machine learning system trained on real-world bank marketing data to predict term deposit subscription probability.

DepositIQ is a supervised ML classification system built on the UCI Bank Marketing Dataset — containing over 45,000 records from a Portuguese bank's direct marketing campaigns. The model predicts whether a client will subscribe to a term deposit based on demographic and financial attributes, empowering banks to optimize campaign targeting and maximize ROI.

How It Works
1

User Input Collection

Client data (age, balance, job, marital status, loans, call duration) is collected via the web form and sent as a POST request.

2

Feature Engineering

The raw form data is converted to a Pandas DataFrame. Default values are injected for unseen features to match the training schema.

3

Pipeline Inference

The serialized model.pkl pipeline applies preprocessing (encoding + scaling) and runs the classifier to output a binary prediction.

4

Result Rendering

The prediction (Yes/No subscription) is returned to the UI with contextual messaging, styled to communicate confidence clearly.
