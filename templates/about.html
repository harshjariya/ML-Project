ℹ️ About the Project

Bank Marketing Prediction is a Machine Learning web application designed to help banks predict whether a client will subscribe to a term deposit. This targeted marketing strategy helps optimize promotional campaigns and allocate resources effectively.

How It Works

By analyzing historical marketing data, the deployed machine learning model predicts client behavior based on specific demographic and financial attributes using advanced classification algorithms.

Key Input Features
  • Demographics: Age, Job, Marital Status
  • Financials: Account Balance, existing Loans
  • Campaign Data: Call Duration
